# QnA Hub Frontend

A modern frontend application for a Question & Answer platform where users can ask questions, explore discussions, and interact with the community.

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** CSS / Tailwind CSS
- **State Management:** React Hooks / Context API / Tanstack Query
- **API Integration:** REST APIs

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/YOUR-USERNAME/qnaHub-fe.git
cd qnaHub-fe
```

### 2. Install dependencies

```
npm install
```

### 3. Run the development server

```
npm start
```

The app will run on:

```
http://localhost:3000
```
